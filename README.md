## Descripción

Este es un proyecto de podcast interactivo diseñado para la asignatura de Interfaces de Usuario (RA4). Se enfoca en proporcionar una experiencia visual moderna y una navegación fluida para el usuario.

## Accesibilidad

El sitio ha sido desarrollado siguiendo principios de diseño accesible, incluyendo:

- Navegación completa mediante teclado.
- Uso de etiquetas semánticas HTML5.
- Contrastes de color optimizados para una mejor legibilidad.

## Edición Audiovisual

Tanto las pistas de audio como el montaje de video de este podcast han sido editados íntegramente utilizando **DaVinci Resolve**, garantizando una postproducción profesional en todo el material multimedia.
